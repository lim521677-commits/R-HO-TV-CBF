The files plot_att_disturbance, plot_att_error, plot_control, plot_po_disturbance, plot_po_error, and plot_quav can display the simulation results only after running UAVbackstepping.mdl.

The files plot_dynamic_distance, plot_static_distance, and plot_quav_save require running UAVbackstepping.mdl first, and then executing save_UAV_posdata, save_desired_traj, save_distance, and save_obstacle in sequence to generate the corresponding simulation results
